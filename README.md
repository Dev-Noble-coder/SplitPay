# SplitPay

**SplitPay** is a digital contribution and rotational savings platform designed to help students and small communities automate group savings and payouts. 

The platform modernizes the traditional “ajo” (or Esusu/contribution) system by allowing users to create trusted savings circles. Members contribute fixed amounts daily, weekly, or monthly, while the system automatically handles tracking, reminders, and payout rotations.

---

## 💡 How It Works (Example)

1. **7 students** create a circle and agree to contribute **₦500 daily**.
2. The pooled contribution becomes **₦3,500**.
3. One member receives the payout based on the automated contribution rotation.
4. The process continues seamlessly until every member has received their turn.

## 🎯 The Problem We Solve
SplitPay removes the stress of manual contribution tracking, payment disputes, fake transfer proofs, and unreliable group members by providing a transparent, automated, and secure system.

## 🚀 Core Features
- **Savings Circles:** Create and join contribution circles easily.
- **Automated Payout Rotation:** Fair and automated scheduling for payouts.
- **Payment Tracking:** Full visibility into contribution history.
- **Digital Wallet:** Built-in wallet funding and withdrawal capabilities.
- **Smart Reminders:** Push notifications and reminders to keep members on track.
- **Trust System:** Trust scores and member verification to ensure reliability.
- **Easy Invites:** Invite system via links or codes.
- **Dashboard:** Real-time contribution dashboard.
- **Group Management:** Defaulter detection and active group management tools.

## 🛠️ Tech Stack
This project is built using modern web technologies, focusing strictly on a mobile-first user experience.
- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Iconsax React](https://iconsax.io/)
- **Typography:** Outfit (Google Fonts)

## 📱 Mobile-First Experience
To maintain a focused and premium application feel similar to modern Fintech apps, SplitPay is strictly optimized for mobile devices. Desktop environments will gracefully prompt the user to switch to a mobile device for the best experience.
